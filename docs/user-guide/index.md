# 用户指南

panel-flow 七步 AI 驱动工作流，将你的故事转化为专业级视频内容。

## 工作流程

1. [导入与分析](./import-analysis) — 上传小说/剧本/提示词
2. [AI 分析](./workflow-overview) — 多模型并行内容分析
3. [脚本生成](./script-generation) — AI 生成结构化脚本
4. [分镜设计](./storyboard-design) — 自动生成分镜图
5. [角色设计](./character-design) — AI 角色一致性
6. [批量渲染](./rendering-export) — 多模型并行渲染
7. [导出](./rendering-export) — 导出 MP4/WebM/MOV
